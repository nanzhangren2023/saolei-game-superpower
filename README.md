# 扫雷游戏

一个基于纯静态前端实现的扫雷游戏，支持多主题切换和完整的测试覆盖。

## 功能特性

- **三种难度**：初级（9x9，10 雷）、中级（16x16，40 雷）、高级（16x30，99 雷）
- **五套主题**：深色经典、Windows 经典、太空探险、海洋世界、糖果派对
- **主题持久化**：使用 localStorage 保存用户偏好
- **首次点击保护**：第一次点击永远不会踩雷
- **完整测试**：单元测试分支覆盖率 >95%，含 E2E 测试覆盖

## 快速开始

### 安装依赖

```bash
npm install
```

### 运行游戏

直接在浏览器中打开 `index.html` 即可。

### 运行测试

```bash
# 运行单元测试（带覆盖率）
npm test -- --coverage

# 运行 E2E 测试
npm run test:e2e
```

## 项目结构

```
项目根目录/
├── docs/                  # 文档目录
│   └── superpowers/
│       └── specs/         # 设计规范
├── src/                   # 源代码目录
│   ├── css/               # 样式文件
│   │   ├── base.css       # 基础重置样式
│   │   ├── themes.css     # 主题 CSS 变量
│   │   └── components.css # 使用变量的组件样式
│   └── js/                # JavaScript 文件
│       ├── Minesweeper.js # 扫雷核心逻辑
│       ├── renderer.js    # 渲染与事件绑定
│       ├── theme-manager.js # 主题管理器
│       └── *.test.js      # 单元测试
├── e2e/                   # E2E 测试
│   ├── pages/             # 页面对象
│   └── tests/             # E2E 测试用例
├── index.html             # 入口文件
└── package.json           # 依赖与脚本配置
```

## 主题系统

| 主题键 | 显示名称 | 特色 |
|--------|---------|------|
| dark | 深色经典 | 默认深蓝/紫渐变风格 |
| classic | Windows 经典 | 怀旧 Windows 95 风格（3D 边框） |
| space | 太空探险 | 科幻深空风格（紫罗兰强调色） |
| ocean | 海洋世界 | 清新蓝色海洋风格 |
| candy | 糖果派对 | 可爱粉色/彩虹风格 |

## 开发规范

- 所有非代码文件使用中文编写
- 代码注释可以使用中文或英文
- 测试文件与源文件平级放置（`*.test.js` 与源文件相邻）
- 单元测试分支覆盖率要求 >= 95%
