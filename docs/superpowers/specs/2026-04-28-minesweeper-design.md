# 扫雷游戏设计规范

## 项目概述

- **项目名称**：浏览器扫雷游戏
- **类型**：单页面网页游戏
- **核心功能**：经典扫雷游戏，可在网页浏览器中游玩
- **目标用户**：寻找快速、怀旧解谜体验的休闲玩家

---

## UI/UX 设计规范

### 布局结构

**页面区域**：
1. **顶部区域** - 游戏标题和难度选择器
2. **游戏面板** - 包含地雷网格、计时器和地雷计数器
3. **底部区域** - 游戏说明或版权信息

**响应式断点**：
- 桌面端：完整尺寸网格，居中布局
- 平板/移动端：网格自适应屏幕宽度（最小 320px）

### 视觉设计 - 玻璃拟态风格

**色彩方案**：
- 背景：从 `#1a1a2e` 到 `#16213e` 的线性渐变
- 玻璃方块（未翻开）：`rgba(255, 255, 255, 0.1)` 配合背景模糊
- 玻璃方块（已翻开）：`rgba(255, 255, 255, 0.05)`
- 强调色：`#00d4ff`（青色光晕）
- 危险色：`#ff6b6b`（地雷）
- 成功色：`#4ecdc4`（旗帜）

**数字颜色**：
- 1: `#4fc3f7`（浅蓝）
- 2: `#81c784`（绿色）
- 3: `#e57373`（红色）
- 4: `#7986cb`（靛蓝）
- 5: `#ffb74d`（橙色）
- 6: `#4dd0e1`（青色）
- 7: `#ba68c8`（紫色）
- 8: `#90a4ae`（灰色）

**字体排版**：
- 字体：`'Segoe UI', 'SF Pro Display', sans-serif`
- 标题：28px，粗体，`#ffffff`
- 格子内数字：18px，粗体
- UI 文字：14px，`#b0b0b0`

**间距**：
- 网格格子大小：32px × 32px
- 格子间距：2px
- 面板内边距：20px
- 格子圆角：4px
- 面板圆角：12px

**视觉效果**：
- 玻璃效果：`backdrop-filter: blur(10px)`
- 盒子阴影：`0 8px 32px rgba(0, 0, 0, 0.3)`
- 格子悬停光晕：`box-shadow: 0 0 15px rgba(0, 212, 255, 0.3)`
- 地雷图标：💣 红色光晕
- 旗帜图标：🚩 绿色光晕

### 组件

**1. 难度选择器**
- 下拉菜单或按钮组
- 选项：初级 (9×9)、中级 (16×16)、高级 (16×30)
- 激活状态：青色边框光晕

**2. 游戏标题区**
- 标题："扫雷"
- 当前难度显示

**3. 游戏面板**
- **地雷计数器**：LED 风格显示剩余未插旗地雷数量
- **笑脸按钮**：点击重新开始；显示不同表情（😊 进行中，😵 失败，😎 胜利）
- **计时器**：LED 风格显示已用秒数（最大 999）

**4. 地雷网格**
- 根据难度动态调整大小
- 格子状态：
  - 未翻开：玻璃效果，细边框
  - 已翻开（安全）：较深玻璃，显示数字或空白
  - 已插旗：显示旗帜图标带光晕
  - 问号：可选，显示 ❓
  - 爆炸地雷：红色背景，显示 💣

**5. 游戏结束弹窗**
- 半透明遮罩
- "游戏结束" 或 "你赢了！" 消息
- "再玩一次" 按钮

## 功能设计规范

### 核心功能

1. **网格生成**
   - 根据选择的难度生成空网格
   - 随机放置地雷（确保首次点击安全）
   - 计算所有格子的邻近地雷数量

2. **格子交互**
   - 左键点击：翻开格子
     - 如果是地雷：游戏结束，显示所有地雷
     - 如果是数字：显示数字
     - 如果是空白 (0)：递归翻开所有相邻空白格子
   - 右键点击：切换格子插旗状态（也可以切换问号）

3. **首次点击保护**
   - 首次点击永远不会踩雷
   - 如果首次点击落在地雷上，将地雷移动到第一个可用的空白格子

4. **胜利检测**
   - 当所有非地雷格子都被翻开时获胜
   - 胜利时自动为剩余地雷插旗

5. **计时器**
   - 首次点击时开始计时
   - 胜利或失败时停止
   - 以秒显示（最大 999）

6. **地雷计数器**
   - 显示：总地雷数 - 已插旗格子数
   - 如果插旗过多可以为负数

7. **笑脸按钮**
   - 点击重新开始当前游戏
   - 根据游戏状态显示不同表情

### 用户交互

| 操作 | 效果 |
|------|------|
| 左键点击未翻开的格子 | 翻开格子 |
| 右键点击未翻开的格子 | 切换插旗 |
| 右键点击已插旗的格子 | 切换问号 |
| 点击笑脸按钮 | 重新开始游戏 |
| 选择难度 | 使用新网格开始新游戏 |

### 边界情况

- 点击已翻开的数字格子：无动作
- 点击已插旗的格子：无动作（防止意外翻开）
- 计时器溢出：999 秒后显示 "999"
- 插旗数量超过地雷数量：计数器显示负数（允许）

## 验收标准

1. ✅ 游戏以默认难度（初级）加载
2. ✅ 三个难度级别都能正常工作
3. ✅ 首次点击总是安全的
4. ✅ 左键点击正确翻开格子
5. ✅ 右键点击正确切换插旗
6. ✅ 空白格子级联翻开正常工作
7. ✅ 计时器在首次点击时开始，游戏结束时停止
8. ✅ 地雷计数器正确更新
9. ✅ 胜利条件被正确检测并显示
10. ✅ 失败条件被正确检测，所有地雷显示
11. ✅ 笑脸按钮重新开始游戏
12. ✅ 应用玻璃拟态视觉风格
13. ✅ 响应式布局在不同屏幕尺寸下正常工作

## 技术实现

- **技术栈**：原生 HTML、CSS、JavaScript（无框架）
- **目录结构**：
  ```
  项目/
  ├── index.html      # 主游戏页面
  ├── css/
  │   └── style.css   # 所有样式
  └── js/
      └── game.js     # 游戏逻辑
  ```
- **数据存储**：无需存储（无状态游戏）
- **浏览器支持**：现代浏览器（Chrome、Firefox、Safari、Edge）
